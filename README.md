# Civic AI Dashboard - Electoral Intelligence

## PromptWars Submission

This project transforms the initial Civic Awareness Assistant into a high-end **Electoral Intelligence Dashboard**. It is designed to provide users with accurate, contextual, and timely information regarding the 2026 local elections, voter registration, and student scholarships.

### 🌟 High-End Features
- **Government/Civic Dark Mode Elements**: A professional slate (`#0f172a`) background accented natively with **National Blue** (`#1e40af`) and **Success Green** (`#10b981`).
- **Dynamic Context Engines**: Parses natural language to automatically extrapolate user age, occupation, and location parameters.
- **Actionable Sidebar Widgets**:
  - 📍 **Polling Station Finder**: Direct mapping integration.
  - 📝 **Voter Registration (Form 6)**: Visual checklists for first-time voters.
  - ⏳ **Election Timelines**: Active countdown and ticker for upcoming 2026 local and state polls.
  - 🎓 **Student Civic Hub**: Dynamic tracking for context-specific initiatives like SSP Scholarships and Yuva Nidhi.

### 🤖 Powered by Gemini 1.5 Pro
The underlying natural language classification and response logic is driven by prompt instructions utilizing **Gemini 1.5 Pro**.
* **Simulated Google Search Grounding:** Ensures users receive exact, real-time deadlines (e.g., *May 31, 2026 SSP extension*).
* **Myth-Buster Protocol**: Actively clarifies misinformation and redirects users to official Election Commission of India (ECI) channels.
